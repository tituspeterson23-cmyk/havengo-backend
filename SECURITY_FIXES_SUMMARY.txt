Security Fixes Applied to HavenGo Backend
========================================

Date: $(Get-Date -Format yyyy-MM-dd)

Vulnerabilities Addressed:
- High Severity: Nodemailer (email library)
  * Fixed by upgrading from ^6.10.1 to ^8.0.9
  * Addressed: Email to unintended domain, DoS via recursive calls, SMTP command injection

Remaining Vulnerabilities (8 moderate):
- All related to uuid library in transitive dependencies of:
  * @google-cloud/storage
  * gaxios
  * google-gax
  * teeny-request
  * retry-request
  * firebase-admin (indirectly)

These remain because fixing them would require upgrading firebase-admin to 10.3.0, which is a breaking change and may affect the application's functionality with Firebase services.
